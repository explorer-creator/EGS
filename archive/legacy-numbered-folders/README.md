# 历史编号目录合并说明

以下内容由根目录迁入，避免与主工程 `proxy/`、`front/` 并列混淆：

- **02-proxy-java**：零散的 `LlmController` / `QwenProperties` Java 片段（非完整 Maven 工程）
- **02-proxy-resources**：`application.yml` 等配置样例
- **03-front-reference**：`vite.config.js`、`RobotAgent.vue` 参考片段

**请以仓库内正式路径为准：**

- Java：`proxy/src/main/java/...`
- 前端：`front/vite.config.js`、`front/src/components/RobotAgent.vue`
