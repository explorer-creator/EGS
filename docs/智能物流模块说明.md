# 智能物流模块说明

## 入口

前端顶部导航 **「智慧物流」**，或全局搜索：智慧物流、物流、仓储、WMS、分拣、路径、AGV、库位、入库、出库。

## 功能概览

| Tab | 说明 |
|-----|------|
| **WMS 仓储管理** | 仓库 → 库区（原料/成品/分拣暂存）→ 库位库存；支持入库、出库与流水记录；数据持久化在浏览器 `localStorage` |
| **分拣与波次** | 生成波次任务、拣货站分配、待分拣/进行中/已完成状态流转 |
| **路径规划（A*）** | 网格地图可编辑障碍；设置起点、终点、多停靠点；**计算 A* 路径**；**多停靠点规划**（最近邻顺序 + 段间最短路径合并） |

## 技术说明

- 路径算法：`front/src/utils/pathfinding.js`（网格曼哈顿启发式 A*，与无权重最短路径一致）
- 页面：`front/src/views/SmartLogisticsPage.vue`

## 演示建议

1. WMS：演示入库/出库后库存与状态（空/正常/满）变化  
2. 分拣：新建波次 → 开始分拣 → 完成  
3. 路径：画障碍 → 设起终点 → 计算路径；再加停靠点 → 多停靠点规划
