## 二、近年总量与趋势（文献与新闻口径摘要）

- **货物贸易**：中国连续多年位居全球货物贸易**第一大国**；出口以**机电、高新技术、装备制造**为支柱，进口侧**能源资源、机电中间品、农产品与消费品**占重要份额。  
- **贸易方式**：**一般贸易**占比持续提升，反映国内产业链与品牌出海能力增强；**加工贸易**仍占一定比重，体现全球价值链分工。  
- **区域合作**：与 **东盟** 互为重要贸易伙伴；与 **欧盟、美国、日韩** 贸易往来规模大；「**一带一路**」沿线国家贸易占比稳步上升（具体比例以当年官方发布为准）。

*注：美元计价总值受汇率与价格因素影响，同比解读需结合海关总署说明。*

## 三、商品结构（概念性分类，非 HS 逐章）

以下为**政策与行业分析中常见**的大类划分，便于与饼图对应：

| 大类 | 出口侧含义（简述） | 进口侧含义（简述） |
|------|-------------------|-------------------|
| 机电与高新技术 | 机械设备、电气设备、电子信息、精密仪器等 | 关键零部件、芯片、高端装备等 |
| 能源与矿产 | 少量能源类出口 | 原油、天然气、铁矿石、铜矿砂等 |
| 纺织服装轻工 | 服装、鞋靴、箱包、家具等 | 纺织原料、消费品等 |
| 化工与材料 | 化学品、塑料橡胶制品等 | 有机化学品、塑料、医药原料等 |
| 农产品食品 | 水产品、蔬菜、劳动密集型加工品等 | 大豆、肉类、乳品、水果等 |
| 其他 | 未单列部分 | 未单列部分 |
