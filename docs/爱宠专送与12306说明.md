# 爱宠专送模块说明

## 功能入口

顶部导航 **「🐾 爱宠专送」**，或全局搜索：爱宠、宠物、12306、铁路、车次、托运等。

## 与 12306 的关系

- **中国铁路客户服务中心**（12306.cn）**未向第三方提供**与官网完全一致的开放票务/托运 **REST API** 供公众随意调用。
- 本模块在 **代理服务** 中实现 **`/api/pet-rail/query`**，基于：
  - 主要城市车站间 **球面距离 × 绕行系数** 估算营运里程；
  - 按车型（G/D/K/Z）设定 **参考均速**，计算 **历时**；
  - 按里程与浮动系数估算 **二等座参考票价**；
  - 按体重与里程给出 **宠物托运参考费用**（演示公式）。
- 返回的 **车次号、时刻** 为 **同一出发到达与日期下可复现的演示数据**，**非** 12306 实时余票与官方公布时刻表。

## API

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/pet-rail/stations` | 支持的车站列表 |
| GET | `/api/pet-rail/query?fromStation=&toStation=&date=YYYY-MM-DD&petWeightKg=` | 参考车次、历时、费用 |

## 政策提示（业务展示用）

- **高铁、动车**：多数车次 **不办理** 随车宠物托运；模块中已用文字提示。
- **普速列车**：部分车次可办理托运，**以出发站与 12306 公告为准**。

## 正式接入 12306 数据

若将来有合规数据源或官方合作接口，可替换 `PetRailController` 内生成逻辑，保留前端页面与表格结构。
