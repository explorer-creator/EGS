# 地图与运输路线

## 功能

- **城际运时预估**：选择「我的城市」「目标城市」与运输方式（公路 / 铁路 / 航空），按数学模型估算**运送时间**（球面距离 × 绕行系数 ÷ 平均速度 + 装卸集散时间），并在地图上标注起终点与两市连线  
- **运输地图**页面：OpenStreetMap 底图 + 厂房 / 仓库 / 物料点标注  
- **与您的距离**：浏览器定位后，按 Haversine 计算与各设施**直线距离**并排序  
- **驾车路线**：服务端代理**高德路径规划**（`/api/map/driving`），在地图上绘制折线；未配置 Key 时降级为**直线示意**

### 运时模型说明（前端 `utils/deliveryEstimate.js`）

- 两市球面距离 \(D_0\)：Haversine  
- 等效运输距离 \(D_{\mathrm{eff}} = D_0 \times\) 绕行系数（公路约 1.38，铁路约 1.12，航空约 1.02）  
- 在途时间 \(T_{\mathrm{transit}} = D_{\mathrm{eff}} / v\)  
- 总时间 \(T = T_{\mathrm{transit}} + T_{\mathrm{handling}}\)（装卸、货站、安检等环节，小时）  

参数为演示量级，非精确物流调度。

## 高德 Key 配置（可选）

1. 登录 [高德开放平台](https://console.amap.com/)，创建应用，开通 **Web 服务**  
2. 在 `proxy/src/main/resources/application.yml` 或环境变量中设置：

```yaml
amap:
  key: 你的Web服务Key
```

或：

```bash
set AMAP_KEY=你的Key
```

3. 重启代理。前端可访问 `GET /api/map/config`，`amapConfigured: true` 表示已生效。

## API 说明（代理）

| 方法 | 路径 | 说明 |
|------|------|------|
| GET | `/api/map/config` | 是否已配置高德 Key |
| GET | `/api/map/driving?origin=lng,lat&destination=lng,lat` | 驾车路径，返回 `path: [[lng,lat],...]`、`distance`(米)、`duration`(秒) |

## 设施坐标

默认数据在 `front/src/data/mapFacilities.js`，可按实际厂区修改经纬度。

## 隐私说明

「定位我的位置」使用浏览器 **Geolocation**，需用户授权；拒绝或失败时使用地图中心附近参考点。
